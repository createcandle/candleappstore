How to use Candle App Store

Most privacy and security issues around smart homes are caused by wifi-enabled devices. These form a juicy target for hackers, since they can autonomously access the internet, and be a stepping stone to taking over other devices in your network.

It's highly recommended to avoid using smart devices that connect via Wifi. However, in some cases you will have little choice. Security cameras, for example, need the high bandwidth that wifi provides. If you have a lot of wifi-based smart home devices it's good practice to create a separate wifi network for your wifi-enabled smart home devices.

Recommendations

  • It is highly recommended to plug in a network cable while using this addon.

Advantages

  • You can learn about all the connections that wifi devices make. You may be shocked to learn how they behave.
  • If someone steals your doorbell they may learn your wifi password. In this case, they'll only have the password to this separate network.

Limitations

The wireless chip in your raspberry Pi is much less powerful than those found in dedicated wireless routers. Because of this there are some limitations to be aware of.

  • If you are using Candleappstore without a network cable, then there is less bandwidth available, since what's available has to be shared in two directions (to the real router, and to the connected devices).
  • You can't connect as many devices. If you want to connect multiple devices that require a lot of data transmission, such as network video cameras, then you may want to setup your real router to create a secondary network instead.
